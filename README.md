## Getting Started

Each lab folder contains its own README with specific setup instructions. General steps:

```powershell
# Clone the repository
git clone https://github.com/EruzaDev/cccs106-projects.git
cd cccs106-projects

# Navigate to a lab folder
cd mod6_labs

# Create and activate virtual environment
python -m venv venv
.\venv\Scripts\Activate.ps1

# Install dependencies
pip install -r requirements.txt

# Run the app
python main.py
```
